LAYOUT MANAGER v0.1
-------------------
Open layout-manager.html from the same website as the display.

Current capabilities:
- Drag panels on the five-column, two-row canvas
- Set a panel to one or two columns and one or two rows
- Show or hide panels
- Change title color and title size
- Turn 2px top/right/bottom/left panel borders on or off
- Adjust the Parish Life slideshow width
- Preview changes immediately
- Save automatically in the browser
- Export/import the layout as JSON
- Reset to the RC2 arrangement

Important v0.1 limitation:
The saved layout is stored in the browser being used. Export the JSON file to move the layout to another device or preserve a backup. Multi-screen publishing and server-side saving belong to later versions.
